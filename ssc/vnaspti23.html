SSC, Online CPOSI (T-I), 2017

  • B
  • C
  • D

आपका जवाब गलत है

Explanation:

          • B
          • C
          • D

                • B
                • C
                • D

                    SSC, CHSL, 2012

                    • A
                    • B
                    • C
                    • D
