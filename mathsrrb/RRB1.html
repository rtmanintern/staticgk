              Q8. यदि pq एक दो-अंको वाली संख्या है, तो pq - qp, निम्नलिखित में से किससे पूर्णतया विभाज्य होगी?
              RRB NTPC 25.01.2021 (Shift-II) Stage Ist

              • A
              • B
              • C
              • D

              आपका जवाब गलत है

              Explanation:
              माना दो अंको वाली संख्या (pq) = 10x + y
              एवं ( qp ) = 10y + x
              तो प्रश्नानुसार,
              pq-qp
              = 10x + y - ( 10y + x )
              = 10x + y - 10y - x
              = 9x - 9y
              = 9 (x - y )
              अतः pq - qp, 9 से पूर्णतया विभाज्य होगी।

                आपका जवाब गलत है

                अतः (n³ –n) 6 से हमेशा विभाज्य होगा।
