Q_401. दी समरूपता के आधार पर नहीं दिया गया शब्द/अक्षर या शब्द युग्म/अक्षर युग्म ज्ञात करें।
कप : क्रोकरी : : पेन : ?

  • A
  • B
  • C
  • D

आपका जवाब गलत है

Explanation:
जिस प्रकार कप, क्रोकरी के अंतर्गत आता है, उसी प्रकार पेन, लेखन-सामग्री के अंतर्गत आता है।


    Q_402. दी समरूपता के आधार पर नहीं दिया गया शब्द/अक्षर या शब्द युग्म/अक्षर युग्म ज्ञात करें।
    प्रोटीन : विकास : : कार्बोहाइड्रेट : ?

    • A
    • B
    • C
    • D

    आपका जवाब गलत है

    Explanation:
    जिस प्रकार प्रोटीन से विकास होता है, उसी प्रकार कार्बोहाइड्रेट से ऊर्जा प्राप्त होती है।


      Q_403. दी समरूपता के आधार पर नहीं दिया गया शब्द/अक्षर या शब्द युग्म/अक्षर युग्म ज्ञात करें।
      तपेदिक : फेफड़ा : : मोतियाबिंद : ?

      • A
      • B
      • C
      • D

      आपका जवाब गलत है

      Explanation:
      जिस प्रकार तपेदिक नामक रोग से फेफड़ा प्रभावित होता है, उसी प्रकार मोतियाबिंद रोग से आंख प्रभावित होती है।


        Q_404. दी समरूपता के आधार पर नहीं दिया गया शब्द/अक्षर या शब्द युग्म/अक्षर युग्म ज्ञात करें।
        घर : किराया : : पूंजी : ?

        • A
        • B
        • C
        • D

        आपका जवाब गलत है

        Explanation:
        जिस प्रकार घर से किराया प्राप्त होता है, उसी प्रकार पूंजी से ब्याज प्राप्त होता है।


          Q_405. दी समरूपता के आधार पर नहीं दिया गया शब्द/अक्षर या शब्द युग्म/अक्षर युग्म ज्ञात करें।
          घाटीमार्ग : दर्रा : : विसर्पण : ?

          • A
          • B
          • C
          • D

          आपका जवाब गलत है

          Explanation:
          जिस प्रकार घाटीमार्ग, दर्रा से संबंधित है, उसी प्रकार विसर्पण डेल्टा प्रदेश से संबंधित है।


            Q_406. दी समरूपता के आधार पर नहीं दिया गया शब्द/अक्षर या शब्द युग्म/अक्षर युग्म ज्ञात करें।
            संगमरमर : स्लेट : : नाइस : ?

            • A
            • B
            • C
            • D

            आपका जवाब गलत है

            Explanation:
            संगमरमर, स्लेट, नाइस एवं क्वार्ट्जाइट सभी काकायांतरित चट्टानें हैं।


              Q_407. दी समरूपता के आधार पर नहीं दिया गया शब्द/अक्षर या शब्द युग्म/अक्षर युग्म ज्ञात करें।
              दृढ़त्वचीय प्राणी (पैकीडर्म) : : ? : ?

              • A
              • B
              • C
              • D

              आपका जवाब गलत है

              Explanation:
              जिस प्रकार हाथी दृढ़त्वचीय प्राणी (पैकीडर्म) वर्ग का है, उसी प्रकार कंगारू मार्सूपीअल (धानी प्राणी) वर्ग के अंतर्गत आता है।


                Q_408. दी समरूपता के आधार पर नहीं दिया गया शब्द/अक्षर या शब्द युग्म/अक्षर युग्म ज्ञात करें।
                बाजार : मांग : : कृषि : ?

                • A
                • B
                • C
                • D

                आपका जवाब गलत है

                Explanation:
                जिस प्रकार बाजार मांग पर आश्रित है, उसी प्रकार कृषि, मानसून पर आश्रित है।


                  Q_409. दी समरूपता के आधार पर नहीं दिया गया शब्द/अक्षर या शब्द युग्म/अक्षर युग्म ज्ञात करें।
                  हंसना : आनंद : : रोना : ?

                  • A
                  • B
                  • C
                  • D

                  आपका जवाब गलत है

                  Explanation:
                  हंसने से आनंद व्यक्त होता है। रोने से शोक व्यक्त होता है।


                    Q_410. दी समरूपता के आधार पर नहीं दिया गया शब्द/अक्षर या शब्द युग्म/अक्षर युग्म ज्ञात करें।
                    लड़की : सुंदर : : लड़का : ?

                    • A
                    • B
                    • C
                    • D

                    आपका जवाब गलत है

                    Explanation:
                    लड़की सुंदर (Beautiful) होती है और लड़का मनोहर (Handsome) होता है।
